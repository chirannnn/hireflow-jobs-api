# HireFlow Jobs API

Backend REST API for a job portal platform.

## Tech Stack

- Node.js
- Express.js
- MongoDB (Mongoose)

## Features (Planned)

- User registration & authentication
- Role-based access (user, employer, admin)
- Job management
- Job applications
- File uploads (resumes)
- Pagination, filtering, sorting
